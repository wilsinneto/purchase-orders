## Description

Application written in Nes.js with docker contains creation, modification, removal and a list of requests. To run the application you need to have docker installed. In addition, you must follow the steps below.

## Installation

```bash
$ npm install
```

## Running the docker

```bash
$ docker-compose up --build
```


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
